Sistema desenvolvido em java para realizar um teste de ansiedade 
baseado em 10 perguntas, onde o usuário deve responder com apenas
sim ou não.
Caso 7 perguntas ou mais  sejam respondidas com sim, o sistema deve
retonar uma mensagem de alerta, se o usuario reponder 5 ou 6 perguntas
com sim, uma mensagem de cuidado será emitida, por fim, se for respondidas
até 4 perguntas com sim, o sistema devolverá uma mensagem positiva.
